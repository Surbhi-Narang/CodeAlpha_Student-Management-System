# Student Management System in C

This project is a menu-driven Student Management System developed in C.

## Features
- Add student records
- Display all records
- Search student by roll number
- Update student record
- Delete student record
- Store data permanently using file handling

## Concepts Used
- Structures
- File Handling
- Functions
- Menu-driven programming

This project is part of my CodeAlpha C Programming Internship.
